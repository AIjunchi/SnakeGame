# 贪吃蛇游戏 (Snake Game)

这是一个使用纯 HTML、CSS 和 JavaScript 构建的经典贪吃蛇游戏。界面现代简洁，玩法原汁原味。

## 🎮 功能特性

- **经典玩法**：控制蛇移动吃食物，随着长度增加难度提升。
- **计分系统**：实时显示当前得分，每吃一个食物 +10 分。
- **响应式设计**：界面居中，适配不同屏幕尺寸。
- **键盘控制**：支持方向键流畅控制。
- **游戏状态**：包含开始、暂停（隐含在未开始状态）和游戏结束提示。

## 🚀 如何开始

1. **克隆或下载**本项目到本地。
2. 直接在浏览器中打开 `index.html` 文件。
3. 点击页面中央的 **“开始游戏”** 按钮即可开始。

## 🕹️ 操作说明

使用键盘方向键控制蛇的移动方向：

- `↑` (ArrowUp)：向上移动
- `↓` (ArrowDown)：向下移动
- `←` (ArrowLeft)：向左移动
- `→` (ArrowRight)：向右移动

## 📂 文件结构

- `index.html`: 游戏主页面，包含 Canvas 画布和 UI 结构。
- `style.css`: 样式文件，负责视觉设计和布局。
- `script.js`: 游戏核心逻辑，包含渲染循环、碰撞检测和事件监听。
- `README.md`: 项目说明文档。

## 🛠️ 技术栈

- HTML5 (Canvas API)
- CSS3 (Flexbox)
- JavaScript (ES6+)
